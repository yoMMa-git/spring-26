## Методология

### 1. Искусственное внесение пропусков

Для демонстрации способности алгоритма обрабатывать пропущенные значения, в датасет было внесено **10% пропусков** случайным образом.

```python
from loosify_dataset import inject_missing_values

df_missing = inject_missing_values(df, missing_ratio=0.1, random_state=42)
```

### 2. Заполнение пропусков

Реализован метод заполнения пропусков на основе эмпирического распределения вероятностей по каждому признаку:

```python
def probabilistic_impute(df: pd.DataFrame, random_state: int | None = None) -> pd.DataFrame:
    rng = np.random.default_rng(random_state)
    df_imputed = df.copy()
    
    for col in df_imputed.columns:
        mask_nan = df_imputed[col].isna()
        n_missing = mask_nan.sum()
        if n_missing == 0:
            continue
        
        values, counts = np.unique(df_imputed[col].dropna(), return_counts=True)
        probs = counts / counts.sum()
        
        sampled_values = rng.choice(values, size=n_missing, p=probs)
        df_imputed.loc[mask_nan, col] = sampled_values
    
    return df_imputed
```


### 3. Редукция дерева (Pruning)

Реализован метод **reduced-error pruning**:

1. Для каждого внутреннего узла проверяется, улучшит ли его замена на листовой узел качество на валидационном наборе
2. Узел заменяется листовым узлом с классом большинства, если точность не ухудшается
3. Процесс выполняется в порядке "снизу вверх" (post-order обход)

### 4. Условия остановы обучения

Разбор листового узла прекращается при выполнении одного из условий:

- Глубина достигла `max_depth` (по умолчанию 10)
- Количество примеров меньше `min_samples` (по умолчанию 5)
- Все примеры принадлежат одному классу (идеальная чистота)
- Не найдено улучшающего разбиения
