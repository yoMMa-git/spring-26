# Лабораторная работа: Реализация дерева решений

## Постановка задачи
Нeобходимо реализовать алгоритм построения бинарного решающего дерева и сравнить его с эталонной реализацией.

## Выполненные этапы

### 1. Реализация алгоритма построения дерева ID3 с критерием Джини
Был реализован алгоритм ID3 и критерий Джини согласно справочным материалам. В реализацию были добавлены гиперпараметры, ограничивающие максимальныю высоту дерева (max_depth) и минимальное количество элементов, при котором можно делить вершину (min_samples_split).

### 2. Реализовать обработку пропущенных значений через оценку вероятности
Обработка пропущенных значений была так же реализована согласно справочным материалам. Для корректного вывода была переписана логика инференса для учета вероятностного распределения ответов модели.

### 3. Реализовать алгоритм редукции дерева
Был реализован дополнительный функционал редукции дерева. Вызывается после в рамках обучения модели, после построения самого дерева. Обрезает дерево согласно справочным материалам

### 4. Сравнение с эталонной реализацией  
Разработанный алгоритм сравнивался с версией после прунинга и эталонной реализацией при тех же гиперпараметрах. Результаты приведены в **Таблице 1**.

**Таблица 1.** Сравнение метрик качества классификации

| Модель                      | Accuracy | Precision | F1     |
|----------------------------|----------|-----------|--------|
| Разработанный алгоритм     | 0.9314   | 0.9314    | 0.9314 |
| Алгоритм с прунингом       | 0.9405   | 0.9408    | 0.9406 |
| Эталонная реализация       | 0.9390   | 0.9399    | 0.9390 |

## Интерпретация результатов

Разработанная реализация с алгоритмом прунинга превосходит обычную версию и эталонную в части метрик при идентичных параметрах. В реализации sklearn нет метода встроенного метода прунинга деревьев, что объясняет разницу между встроенной и реализованной моделью
